# Matrix Row and Column Sum

Aplikasi web sederhana untuk menghitung jumlah setiap baris dan kolom pada matriks menggunakan HTML, JavaScript, dan Bootstrap.

## Fitur

- Input nilai matriks secara langsung.
- Menghitung total setiap baris.
- Menghitung total setiap kolom.
- Tampilan responsif menggunakan Bootstrap.
- Mudah digunakan tanpa database atau backend.

## Teknologi

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Bootstrap

## Struktur Project

```
.
├── matrik.html
├── ppp.html
└── README.md
```

## Cara Menjalankan

1. Clone repository:

```bash
git clone https://github.com/username/matrix-row-column-sum.git
```

2. Masuk ke folder project:

```bash
cd matrix-row-column-sum
```

3. Buka file `matrik.html` atau `ppp.html` menggunakan browser.

## Cara Penggunaan

1. Masukkan angka ke dalam matriks.
2. Klik tombol **Proses**.
3. Sistem akan menghitung:
   - Jumlah setiap baris.
   - Jumlah setiap kolom.
4. Hasil akan ditampilkan pada sel terakhir di setiap baris dan kolom.
